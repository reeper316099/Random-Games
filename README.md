# Random Games Collection

A small collection of browser games, including both single-file games and multi-file projects.

## What's included

### One File Games
- `One File Games/chess.html`
- `One File Games/minesweeper.html`
- `One File Games/ultimate-ttt.html`

### Multi-File Games
- `Multi-File Games/Risk/index.html`
- `Multi-File Games/hotzone-na/index.html`

## New launcher page

Open `index.html` from the repository root to get quick links to every playable game in this repo.

## How to play

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click any game link to start playing.

> Tip: Some browsers apply stricter local file security rules. If a game doesn't load correctly via `file://`, run a local static server and open the pages over `http://localhost`.
